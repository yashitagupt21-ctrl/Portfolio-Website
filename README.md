# Portfolio-Website
This repository contains the source code of my personal portfolio website, built using HTML and CSS.
The goal of this project is to showcase my skills, projects, achievments, and provide an easy way fir people to connect with me.

#Features
-Fully responsive and clean layout
-Modern UI using pure HTML and CSS
-Smooth navigations across sections
-Dedicated section for:
  1. About me
  2. Skills
  3. Projects
  4. Contact
-Light weight and fast loading
-Easy to customize and expand

#Technologies Used
HTML
CSS

No frameworks or external libraries were used - the website is built from scratch.
